# CSVtoTally

CSVtoTally is a modern, premium, and 100% private Progressive Web App (PWA) designed to convert Excel/CSV accounting data into Tally-ready XML formats. 

**Live App**: [https://asr-ajay.github.io/csvtotally/](https://asr-ajay.github.io/csvtotally/)

## Features
- **100% Privacy**: All processing happens locally in your browser. No data ever leaves your computer.
- **Support for All Modes**:
  - Accounting Vouchers (Single/Multiple/Combined)
  - Inventory Vouchers (Simple/Multiple)
  - Master Creation (Ledgers/Stock Items)
- **Tally Master Validation**: Upload your Tally masters (HTML) for real-time header and ledger validation.
- **Installable Desktop App**: Works as a PWA with offline support.
- **Premium Aesthetics**: Glassmorphism design with full mobile responsiveness.

## How to Use
1. Open the application.
2. Select your desired Template Type.
3. Download the sample CSV template and fill in your data.
4. (Optional) Upload your Tally Masters for validation.
5. Drop your CSV file and click "Convert to XML".
6. Download the generated XML and import it into TallyPrime (Alt+O > Transactions).

## Tech Stack
- HTML5 / CSS3 (Glassmorphism, Grid, Flexbox)
- Vanilla JavaScript
- PapaParse (CSV Parsing)
- Service Workers (PWA/Offline support)
